# Быстрая инструкция: Сбор информации с сервера

## Быстрый старт

```bash
# 1. Убедитесь, что скрипт исполняемый
chmod +x scripts/collect_server_info.sh

# 2. Запустите скрипт (с указанием SSH ключа и сервера)
./scripts/collect_server_info.sh ~/.ssh/id_rsa root@188.166.240.56

# 3. Или используйте дефолтные значения
./scripts/collect_server_info.sh
```

## Что нужно перед запуском

1. **SSH ключ** должен быть настроен и иметь доступ к серверу
2. **Проверьте подключение** вручную:
   ```bash
   ssh -i ~/.ssh/id_rsa root@188.166.240.56
   ```

## Результат

После выполнения вы получите файл `server_info_YYYYMMDD_HHMMSS.md` с полной информацией о сервере.
